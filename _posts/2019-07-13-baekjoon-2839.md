## 문제

[문제 링크](https://www.acmicpc.net/problem/2839)

## 어떻게 접근할 것인가?

>* 5로 최대한 나눠주고 3이 나누어 떨어질 때까지 5를 다시 더해주는 식으로 풀었다. 

## 코드

    #include <cstdio>

    int sum, num, left;
    int main(){
        scanf("%d", &num);
        sum = num / 5;
        left = num % 5;
        while(left <= num){
            if(left % 3 == 0){
                sum += left / 3;
                break;   
            }
            sum --;
            left += 5; 
        }
        if(left > num)printf("-1");
        else printf("%d", sum);
    }
