## 문제

[문제 링크](https://www.acmicpc.net/problem/11399)


## 어떻게 접근할 것인가?

>* 총 기다리는 시간을 줄이기 위해서는 먼저 끝낼 수 있는 사람은 먼저 끝내줘야 한다.
>* 때문에 일단 사람을 다 받고 솔팅을 먼저하고 값을 더해줘라

### 코드

    #include <string>
    #include <vector>
    #include <iostream>
    #include <algorithm>

    using namespace std;
    int main() {
        int n, temp;
        int num = 0;
        int answer = 0;
        vector<int> v;
        cin >> n;
        for(int i = 0; i < n; i++){
            cin >> temp;
            v.push_back(temp);
        }
        sort(v.begin(), v.end());
        for(int i = 0; i < n; i++){
            num += v[i];
            answer += num;
        }
        cout << answer << endl;
    }
